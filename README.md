# Volta
Dataset.py
- ใช้เก็บข้อมูลใบหน้าคน 
- เมื่อ Run Dataset.py ให้ใส่ชื่อคนที่ต้องการถ่าย
  กดตัว c เพื่อถ่ายหน้าตามจำนวนที่ต้องการ
  กด esc เพื่อออกจากการถ่ายรูป

Train.py
- เมื่อถ่ายใบหน้าคนเสร็จให้ Run ไฟล์นี้
- ทำหน้าที่ ประมวลผลใบหน้าคนที่ถ่ายมาใหม่แล้วเก็บในรูปไฟล์ .yml

Detect.py
- ทำหน้าที่ตรวจจับใบหน้าคน จะขึ้นชื่อคนเมื่อมีข้อมูลคนๆนั้น จะขึ้น not found เมื่อไม่มีคนๆนั้น
- กด esc เพื่อปิด

Clear.py
- เมื่อ Run จะสามารถลบข้อมูลชื่อคนที่ไม่ต้องการออกได้ หรือ พิมพ์ ALL เพื่อลบทั้งหมด
